How I executed my C code on browser

STEP 1: Installing apacahe web server on Ubuntu

Apache Web Server is a software package that turns a computer into an HTTP server. That is, it sends web pages – stored as HTML files – to people on the internet who request > The first step towards executing a C code on browser is to install Apache on ubuntu.I followed the steps mentioned in the given website:
https://phoenixnap.com/kb/how-to-install-apache-web-server-on-ubuntu-18-04
The commands that i followed were as follows:

To install apache

  • sudo apt-get update
  • sudo apt-get install apache2

For verification

  • Browse: http://local.server.ip

For configuration

  • ifconfig
  • sudo apt install net-tools
  • ifconfig
  • sudo apt install systemd
  • sudo /etc/init.d/apache2 restart
  • sudo ufw enable
  • sudo ufw status

The status was active.

STEP 2: Installing a text editor

After installing Apache on ubuntu,i installed vim text editor by following the given command:

  • sudo apt-get install build-essential vim apache2

I got to know about this command in a tutorial on youtube.

STEP 3: Writing a C code on terminal

After installing vim text editor i wrote a very simple C code to execute Hello world by following the given commands:

  • vim hello.c

And wrote the following C code:

#include<stdio.h>
int main()
{
printf("Content-type: text/plain");
printf("Hello World\n");
return 0;
}

STEP 4: Compiling the C code and executing it

To compile the C code i used the given command:

  • gcc hello.c -o hello
  • ./hello

After this i followed the commands explained in the given tutorials:
https://youtu.be/dKKshPF2OJ4

The CGI (Common Gateway Interface) defines a way for a web server to interact with external content-generating programs, which are often referred to as CGI programs or CGI scripts. It>

The commands were as follows:

  • sudo a2enmod cgi (to enable cgi)
  • sudo service apache2 restart
  • sudo cp hello /usr/lib/cgi-bin

Now search:

  • ip address/cgi-bin/hello

To know my ip address i used the command:

  • hostname -I | awk '{print $1}'

Following the above commands Hello world was exexcuted on the browser
