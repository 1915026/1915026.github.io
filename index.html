How I executed my C code on browser

STEP 1: Installing apacahe web server on Ubuntu

Apache Web Server is a software package that turns a computer into an HTTP server. That is, it sends web pages – stored as HTML files – to people on the internet who request them. It is open-source software, which means it can be used and modified freely.

The first step towards executing a C code on browser is to install Apache on ubuntu.I followed the steps mentioned in the given website:

https://phoenixnap.com/kb/how-to-install-apache-web-server-on-ubuntu-18-04

STEP 2: Installing a text editor

After installing Apache on ubuntu,i installed vim text editor by following the given command:

sudo apt-get install build-essential vim apache2

I got to know about this command in a tutorial on youtube.

STEP 3: Writing a C code on terminal

After installing vim text editor i wrote a very simple C code to execute Hello world by following the given commands:

vim hello.c

And wrote the following code:

#include

int main()

{

printf("Content-type: text/plain");

printf("hello\n");

return 0;

}

STEP 4: Compiling the code and executing it

To compile the c code i used the given command:

gcc hello.c -o hello

./hello

After this i followed the commands explained in the given tutorials:

https://youtu.be/dKKshPF2OJ4

The commands were as follows:

sudo a2enmod cgi

sudo cp hello /usr/lib/cgi-bin

Now search:

ip address/cgi-bin/hello
